Anotaciones sobre JavaScript

Características de JS

  • Lenguaje de alto nivel

  • Isomórfico: se puede utilizar para todas las facetas de un programa (front, back, persistencia de datos)

  • Multiparadigma: puede usarse para programación orientada a objetos, programación estructurada, etc.

  • Interpretado: no necesita fase previa de compilación antes de poder ser ejecutado. Se ejecuta mientras se compila

  • Sensible a mayúsculas y minúsculas: perro != Perro

  • Dinámico

  • Debilmente tipado: las variables no se casan con un tipo de dato

Orden de código

  1. Importación de módulos

  2. Declaración de variables

  3. Declaración de funciones

  4. Ejecución de código

Tipos de datos en JS

  • Primitivos: se accede directamente al valor

    • string

    • number

    • boolean

    • null

    • undefined

    • NaN: not a number

  • Compuestos: se accede a la referencia del valor

    • object: {}

    • array: []

    • function: () {}

    • Class: {}

Métodos que dependen de un constructor

Los métodos que dependen de un constructor son los métodos llamados estáticos
Estos métodos son aquellos que solo pueden usar objetos de esta clase.
