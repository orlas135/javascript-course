Módulos en JavaScript

Aclaraciones y anotaciones

Al querer hacer uso de módulos JavaScript en archivos HTML, se debe especificar en la etiqueta script con el atributo type que el archivo es un módulo.
Dicho atributo debe ser igualado al valor module

No se pueden hacer dos exportaciones por defecto. Solamente una.

Las exportaciones que no se hacen por defecto, se tienen que hacer mediante destructuring.

No se pueden exportar por defecto constantes ni variables ni funciones expresadas en la misma línea de su declaración. Todo esto debido al ordenamiento de código de JS.
Para exportar por defecto variables, constantes y funcions expresadas, primero se declara la misma y, en la línea siguiente, se hace el export por defecto
